
## 简介
xtool 是java 工具辅助类

## 特征&提供
- BeanByRefMapper bean 之间的拷贝
- 集合操作类 Lists，Maps，Sets，Sorts
- 。。。。。。

 
## 快速开始
     maven 直接引入
        <dependency>
              <groupId>com.github.xphsc</groupId>
              <artifactId>xtool</artifactId>
              <version>1.0.0</version>
          </dependency>
